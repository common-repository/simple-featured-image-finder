Simple Featured Image Finder helps you to find and use copyright free images for your posts or pages with ease

== Description ==
[Gif example](http://mentorgashi.com/sfif.gif)

By installing and activating Simple Featured Image Finder, the plugin will add a search field below traditional featured image box usually found on the right column of post/page add/edit form. The search field is a port to thousands of HQ images provided by **Unsplash.com** with copyright free, read the license here: [https://unsplash.com/license](https://unsplash.com/license)

The settings page located below **Settings** menu of Wordpress with label **Simple Featured Image Finder**, this page helps you manage the API credentials of **Unsplash**. You can obtain your own API credentials on [https://unsplash.com/developers](https://unsplash.com/developers) by signing up and creating an APP, afterwards copy the Application ID.

Just pick the best image listed by your search and forget downloading the images and uploading, just click the **Set featured image** button and let the magic happen.

== Installation ==

1. Upload `simple-featured-image-finder` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to Settings > Simple Featured Image Finder
4. Paste your Unsplash API, APP ID and save changes.
Establishing the plugin access to **Unsplash API** requires an APP ID, please navigate to [https://unsplash.com/developers](https://unsplash.com/developers) sign-up for an account and create an APP here: [https://unsplash.com/oauth/applications/new](https://unsplash.com/oauth/applications/new)
After you've created your application copy the Application ID string and navigate to **WP-Admin** > **Settings** > **Simple Featured Image Finder** settings page, paste the long **APP ID** string into the guided field and then hit the Save changes button. That's it, enjoy setting your featured images with ease, thanks to **Unsplash** HQ images.
